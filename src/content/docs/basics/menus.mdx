Create __menus__ by defining entries, localizing each entry, and rendering the resulting data structure.


## Examples

For example, to define entries for the main menu:

```toml title="config/_default/menus.toml"
[[main]]
  name = 'Home'
  pageRef = '/'
  weight = 10

[[main]]
  name = 'Products'
  pageRef = '/products'
  weight = 20

[[main]]
  name = 'Services'
  pageRef = '/services'
  weight = 30

```
