This guide will walk you through the steps to manually update from Hyas v1 to Hyas v2.

## Upgrading

You can upgrade your project by following the steps below.

### 1. Clean dependencies

Clean dependencies currently installed to avoid conflicts.

<Tabs>
  <TabItem label="npm">
```bash
npm run clean:install
```
  </TabItem>
  <TabItem label="pnpm">
```bash
pnpm run clean:install
```
  </TabItem>
  <TabItem label="Yarn">
```bash
yarn run clean:install
```
  </TabItem>
</Tabs>

### 2. Update `package.json`

Replace the contents of your project's `package.json` with the following:

```json
// package.json
{
  "name": "hyas-project",
  "version": "0.0.0",
  "description": "Hyas",
  "author": "Hyas",
  "license": "MIT",
  "scripts": {
    "create": "hugo new",
    "dev": "hugo server --disableFastRender --noHTTPCache",
    "format": "prettier **/** -w -c",
    "build": "hugo --minify --gc",
    "preview": "vite preview --outDir public"
  },
  "engines": {
    "node": ">=20.11.0"
  }
}
```

### 3. Install Hyas, Prettier, and Vite

First, add an `.npmrc` file to your project root with the following:

```ini
// .npmrc
enable-pre-post-scripts = true
auto-install-peers = true
node-linker = hoisted
prefer-symlinked-executables = false
```

#### Install Hyas

Next, install the latest version of Hyas:

<Tabs>
  <TabItem label="npm">
```bash
npm install gethyas@latest
```
  </TabItem>
  <TabItem label="pnpm">
```bash
pnpm install gethyas@latest
```
  </TabItem>
  <TabItem label="Yarn">
```bash
yarn add gethyas@latest
```
  </TabItem>
</Tabs>

#### Install Prettier and Vite

Then, install the latest version of Prettier and Vite â€” as `devDependencies`:

<Tabs>
  <TabItem label="npm">
```bash
npm install -D prettier@latest vite@latest
```
  </TabItem>
  <TabItem label="pnpm">
```bash
pnpm install -D prettier@latest vite@latest
```
  </TabItem>
  <TabItem label="Yarn">
```bash
yarn add -D prettier@latest vite@latest
```
  </TabItem>
</Tabs>

### 4. Install integrations

Optionally, install the the recommended integrations in your project:

<Tabs>
  <TabItem label="npm">
```bash
npm install @hyas/images@latest @hyas/seo@latest
```
  </TabItem>
  <TabItem label="pnpm">
```bash
pnpm install @hyas/images@latest @hyas/seo@latest
```
  </TabItem>
  <TabItem label="Yarn">
```bash
yarn add @hyas/images@latest @hyas/seo@latest
```
  </TabItem>
</Tabs>

:::note[Need to continue?]
After upgrading Hyas to the latest version, you may not need to make any changes to your project at all!

But, if you notice errors or unexpected behavior, please check below for what has changed that might need updating in your project.
:::

## Configuration

Update your configuration files for the recommended integrations:

- [Images](https://images.gethyas.com/docs/start-here/manual-setup/)
- [SEO](https://seo.gethyas.com/docs/start-here/manual-setup/)

## Know Issues

### ENOENT

When you get an `ENOENT` error message, run the following command:

<Tabs>
  <TabItem label="npm">
```bash
npm run clean:install && npm install
```
  </TabItem>
  <TabItem label="pnpm">
```bash
pnpm run clean:install && pnpm install
```
  </TabItem>
  <TabItem label="Yarn">
```bash
yarn run clean:install && yarn install
```
  </TabItem>
</Tabs>
