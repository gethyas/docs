## What is Hyas?

Hyas is an **all-in-one Hugo-npm framework** for building **fast, flexible, and easy-to-use** websites.

## Key features

- **Hugo powered:** A fast and flexible framework for building websites.
- **Project dependencies**: Manage *all* your project's dependencies with npm.
- **Production-ready**: Optimized for speed, SEO, and security.
- **Customizable:** Tailwind, Images, and other integrations to choose from.
- **Development tools:** Find and fix problems in scripts, styles, and markdown.
- **Deploy anywhere:** Deployment guides to different deployment services.

Check out our detailed [Why Hyas](/concepts/why-hyas/) breakdown to learn more about what makes Hyas special. ✨

## Start your first project

Get a new Hyas project up and running locally with our helpful `create-hyas` CLI wizard!

<Tabs>
  <TabItem label="npm">
```bash
# create a new project with npm
npm create hyas@latest
```
  </TabItem>
  <TabItem label="pnpm">
```bash
# create a new project with pnpm
pnpm create hyas@latest
```
  </TabItem>
  <TabItem label="Yarn">
```bash
# create a new project with yarn
yarn create hyas
```
  </TabItem>
</Tabs>

Our [Installation Guide](/install/auto/) has full, step-by-step instructions for installing Hyas using our CLI wizard, creating a new project from an existing Hyas GitHub repository, and for installing Hyas manually.
